# lyricMatch

A real-time multiplayer music trivia game where each round serves up a letter (A→Z) and a random word from that letter's music pool — your job is to type an actual lyric from any song you know that uses it, name the artist, and let Gemini AI decide how credible you really are (0–100% confidence score, no lying to the machine). Points scale by letter rarity (Q and Z hit different), bonus for using the prompt word, bonus for the right artist, and a one-click link drops you into Spotify, YouTube Music, or Apple Music to hear the song for real — because yes, someone will absolutely claim a made-up lyric and the AI will call them out. Covers every genre, era, and language on the planet: pop, hip-hop, R&B, afrobeats, K-pop, Latin, grime, drill, dancehall, Bollywood, and everything in between. Firebase handles auth and the global leaderboard, Socket.io keeps the rooms live, and the word pool can be expanded to 1000+ words per letter using the Grok prompt in `/data/GROK_PROMPT.md`.
